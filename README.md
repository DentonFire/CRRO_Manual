# DFD CRR Officer Operations Manual

A comprehensive pass-down guide and standard operating procedures reference for the Community Risk Reduction Officer position at the Denton Fire Department.

## About

This site documents the full scope of the CRR Officer role — a single FTE responsible for 31 program areas spanning community outreach, public education, technology development, the Civil Service hiring process, accreditation support, pension board service, and administrative reporting. It is designed so that a successor can assume and effectively manage every function of this position.

**Current Version:** 0.2 — March 2026
**Prepared by:** Captain Hunter Lott, Community Risk Reduction Officer

## Site Structure

| Section | Pages | Coverage |

| **Part III: Recruitment & Hiring** | Chapters 8–11 | Civil Service exam, hiring process, candidate pipeline, PAT |
| **Part IV: Administration** | Chapters 12–15 | Budget, weekly/quarterly reporting, accreditation, pension |
| **Part V: Technology** | Chapters 16–17 | Digital platforms, system administration, succession planning |
| **Appendices** | A–I | Quick reference cards, CRR tools docs, newsletter archive, contacts, version history, email templates, vendor reference, CRR annual calendar, recruiting annual calendar |

## Local Preview

No build tools or server required. Clone the repo and open `index.html` in any browser.

```
git clone https://github.com/[your-username]/crro-manual.git
open crro-manual/index.html
```

## Updating Content

Manual source content is maintained as chapter-level Markdown files. To rebuild the site after editing:

```
python3 build_site.py
```

This regenerates all 27 HTML pages from the Markdown source files. Commit and push to deploy via GitHub Pages.

## Access

This is an internal operations document for the Denton Fire Department. It contains operational procedures and institutional knowledge for the CRR Officer position. Handle in accordance with department information security policies.

Sensitive credentials are **not** stored in this repository. Account credentials are maintained separately in the CRR Credentials Workbook (password-protected, CRR Officer OneDrive).
