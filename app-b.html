B.2 Water Shutoff System Architecture

Technology Stack: HTML/JavaScript/Tailwind CSS frontend; Firebase Authentication \+ Cloud Firestore backend; GitHub Pages static hosting; Leaflet.js with OpenStreetMap for mapping; Chart.js for analytics dashboard.

Data Flow: User logs in with @cityofdenton.com email → Firebase verifies credentials → User submits shutoff report via survey form → Data saved to Firestore → Email client opens with pre-filled report → Command Center displays real-time updates.

Email Routing: Water meter shutoffs route to watermetering@cityofdenton.com; Sprinkler system shutoffs route to FirePrevention@cityofdenton.com. CC on all reports goes to the CRR Officer.

Offline Capability: The survey form works without cell service. Data saves locally on the device and automatically syncs when connectivity is restored. A yellow banner indicates offline mode.

B.3 Fire Watch System

The Fire Watch Guidelines tool provides three interactive digital fillable PDF forms for managing fire protection system impairments per International Fire Code (IFC) requirements:

  • Building Representative Notification Form — Notifies building management of impairment
  • Designee Notification Form — Documents the fire watch designee assignment
  • Fire Watch Log — Ongoing documentation of fire watch patrols and observations

The system integrates with the Water Shutoff Survey through the “Fire Watch Issued” checkbox, allowing Command Center tracking of properties with active fire watches.
