Chapter 7

CRR Technology Tools

7.1 Tools Portfolio Overview

The CRR Officer developed and maintains a suite of four web-based operational tools, all accessible via the DFD HUB (SharePoint). These tools support field operations, fire prevention, and community service delivery. All tools are hosted on GitHub Pages at no ongoing cost.

Tool Purpose URL
Water Shutoff Survey Log sprinkler/water meter shutoffs with GPS, damage reporting, automated emails dentonfire.github.io/Water-Shutoff/
Fire Watch Guidelines Interactive guide with 3 digital PDF forms for fire system impairments dentonfire.github.io/Fire-Watch/
Knox Box Guide IFC Section 506 compliance, purchasing, installation, gate access dentonfire.github.io/Knox-Box/
After the Fire Recovery guide for fire victims: housing, utilities, documents, financial, mental health dentonfire.github.io/After-the-Fire/

HUB Access: codenton.sharepoint.com/sites/DentonFire/SitePages/CRR\_Resources.aspx

7.2 Water Shutoff Survey — Detailed Reference

The Water Shutoff Tracking System is the most complex CRR tool and requires the most ongoing maintenance attention. It consists of three components:

Architecture

  • Frontend: HTML, JavaScript, Tailwind CSS
  • Backend: Firebase (Authentication \+ Firestore Database)
  • Hosting: GitHub Pages (static hosting, free)
  • Maps: Leaflet.js with OpenStreetMap tiles
  • Charts: Chart.js

Firebase Project: denton-fire-tools

Firebase Console: console.firebase.google.com

GitHub Repository: github.com/dentonfire/Water-Shutoff

Components

Component URL Purpose
Command Center dentonfire.github.io/Water-Shutoff/ Admin dashboard: real-time map, analytics, CSV export, follow-up tracking
Field Survey Form dentonfire.github.io/Water-Shutoff/water\_shutoff.html Mobile form for field crews: unit selection, address, GPS, system type, damage
Fire Watch Guide dentonfire.github.io/Fire-Watch/ Interactive guide with 3 fillable PDF forms for fire system impairments

Email Routing

When a report is submitted, an email is automatically generated and routed to the appropriate department:

  • Water Meter shutoffs → watermetering@cityofdenton.com (Water Distribution Department)
  • Sprinkler System shutoffs → FirePrevention@cityofdenton.com (Fire Prevention Division)
  • CC on all reports → hunter.lott@cityofdenton.com (CRR Officer)

Security & Access

  • Only @cityofdenton.com email addresses can create accounts
  • Email verification required before write access
  • Firebase security rules enforce authentication
  • Session expires after 30 days of inactivity
  • All data transmitted over HTTPS

Purge Password: ADMIN (used to clear all records from Command Center)

WARNING: The purge function permanently deletes ALL records. Use with extreme caution and only when necessary.

Key Code Locations

Updating the Code

Code changes can be made directly through the GitHub web interface (easiest) or via Git command line:

  1. Go to github.com/dentonfire/Water-Shutoff
  1. Click on the file to edit
  1. Click the pencil icon (Edit this file)
  1. Make changes
  1. Click "Commit changes"
  1. Changes go live within 1-2 minutes via GitHub Pages auto-deployment

[ACTION REQUIRED] New CRR Officer needs: (1) GitHub access to the dentonfire organization, (2) Firebase console access to the denton-fire-tools project, (3) Update the CC email address in both HTML files to the new Officer’s email.

Offline Capability

The Water Shutoff Survey works offline. When field crews have no cell signal, the form can still be filled out and submitted. Data saves locally to the device and automatically syncs when connectivity is restored. A yellow banner appears when offline to indicate status.

7.3 Fire Watch Guidelines

The Fire Watch Guidelines tool provides interactive guidance for managing fire protection system impairments per International Fire Code (IFC) requirements. It includes three digital fillable PDF forms:

  • Building Representative Notification Form
  • Designee Notification Form
  • Fire Watch Log

These forms are used by Fire Prevention staff when fire suppression systems are shut off in occupied buildings. The system tracks which properties have active fire watches and enables follow-up on extended impairments.

7.4 Knox Box Guide

The Knox Box Guide provides comprehensive property access guidance covering IFC Section 506 requirements, purchasing procedures, installation specifications, and gate access protocols. This tool serves both property owners/managers and fire department personnel with standardized procedures for Knox Box implementation.

7.5 After the Fire

The After the Fire guide is a comprehensive recovery resource for fire victims. It covers immediate safety, housing assistance, utility restoration, document replacement, financial recovery resources, and mental health support. The guide features a mobile-friendly interface with QR code access and bookmark-to-phone instructions.

Content was developed in coordination with City of Denton Building Safety. The Building Safety section covers the reconstruction process including building permits, courtesy fire/disaster inspections, contractor registration, and inspection requirements. Key contact: Building Safety at 940-349-8600.

Physical handouts were also ordered for non-phone access, enabling distribution at fire scenes by Operations crews.
